# JLPT N4 Grammar (Bunpou)

Source-of-truth catalogue for N4 grammar patterns. Build pipeline (`tools/build_data.py`) parses this file into `data/grammar.json`. Schema per §12 / §28 / Appendix A.1 of `procedure-manual-build-next-jlpt-level.N4.md`.

Each pattern entry under one of the 18 category sections. Entry format:

```
### <pattern> - <id>
**Meaning EN:** <one sentence>
**Meaning JA:** <one sentence>
**Tier:** core_n4 | late_n4 | n3_borderline | n5_prerequisite
**Form rules:** <attaches_to + conjugations>
**Examples:**
- ex1: <ja> - <en>
- ex2: <ja> - <en>
**Common mistakes:**
- wrong: <bad form>
  right: <good form>
  why: <one-sentence reason>
**Notes:** <optional cross-references>
```
